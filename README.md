## Project Overview
This Project presents an Interactive Amazon Sales Analysis Dashboard created in Microsoft Excel. It provides insights into sales performance, order status,
payment methods, monthly trends and state-wise sales using Pivot Tables, Pivot Charts and Slicers.

## Objective
To analyze Amazon sales data and create an interactive dashboard that helps track key business metrics and identify sales trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Dashboard Features
- Total Sales
- Total Orders
- Total Quantity
- Average Order Value
- Sales by Year
- Payment Method Analysis
- Order Status Analysis
- State-wise Sales
- Monthly Sales Trend

## Key Insights
- Maharashtra recorderd the highest sales.
- Piggy coin bank generated the highest revenue.
- Sales peaked during December.
- Delivered orders significantly exceeded returned orders.
- Customers preferred prepaid payments over Cash on Delivery(COD).

## Dataset
The Excel workbook contains the raw data, cleaned data, pivot tables, formulas and the interactive dashboard used in this project.
